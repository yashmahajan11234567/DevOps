# Assignment 5 - Express.js Application

## Overview
A Node.js and Express.js-based web application with integrated frontend and backend components.

## Project Structure
```
ASS5/
├── Outputscreenshots/  # Application screenshots
└── Sourcecode/
    ├── app.js          # Express application configuration
    ├── server.js       # Server entry point
    ├── package.json    # Dependencies and scripts
    ├── index.html      # Main HTML file
    └── [other frontend files]
```

## Features
- Express.js server for handling requests
- Integrated HTML/CSS/JavaScript frontend
- RESTful API routing
- Modular application structure

## Technology Stack
- **Backend:** Node.js, Express.js
- **Frontend:** HTML5, CSS3, JavaScript
- **Runtime:** Node.js

## Installation & Setup
1. Navigate to the `Sourcecode` directory
2. Install dependencies: `npm install`
3. Start the application: `node server.js` or `npm start`
4. Open your browser to the server URL (check console for exact port/address)

## Usage
- Access the web interface through the running server
- The application handles both frontend serving and backend API logic
- Check console output for server status and debug information

## Available Scripts
- `npm start` - Start the server (if configured in package.json)
- `node server.js` - Direct server execution
- `node app.js` - Alternative startup method

## Output
Screenshots and visual demonstrations are available in the `Outputscreenshots` directory.
