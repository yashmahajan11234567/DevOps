# Portfolio Requirements

## Project Overview

This project is a professional Data Analytics showcase website created for DevOps Assignment 1.

The purpose is to demonstrate:

- HTML
- CSS
- JavaScript
- Git
- GitHub
- GitHub Pages
- Professional frontend organization

This is NOT a final career portfolio.

The website should represent an aspiring Data Analyst using modern UI/UX principles.
