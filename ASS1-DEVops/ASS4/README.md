# Assignment 4 - Web Application with Server

## Overview
A web application combining static HTML files with a Node.js backend server for dynamic functionality.

## Project Structure
```
ASS4/
├── Outputscreenshots/           # Application screenshots
└── Sourcecode/
    ├── Ass2_part2(FSDL).html   # Full-stack development lab file
    ├── Ass4.html                # Main assignment file
    ├── package.json             # Node.js dependencies
    └── server.js                # Express server
```

## Features
- Static HTML frontend with CSS and JavaScript
- Node.js/Express backend server
- Dynamic server-side processing

## Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js

## Installation & Setup
1. Navigate to the `Sourcecode` directory
2. Install dependencies: `npm install`
3. Start the server: `node server.js` or `npm start`
4. Open your browser and navigate to the provided server URL (typically `http://localhost:3000` or `http://localhost:5000`)

## Usage
- Access the application through the web server
- The server handles requests and serves the HTML files
- Interact with the frontend interface for application functionality

## Output
Screenshots and visual demonstrations are available in the `Outputscreenshots` directory.

## Notes
- Ensure Node.js is installed before running
- Check the server.js file for configuration details and port information
