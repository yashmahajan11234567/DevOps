# Assignment 4B - Timetable Scheduling System

## Overview
This is a comprehensive timetable scheduling application that manages subjects, teachers, and timetable assignments.

## Project Structure
```
ASS_4B/
├── Outputscreenshots/     # Screenshots of the application
└── Sourcecode/            # Source code
    ├── index.html         # Main HTML file
    ├── script.js          # Frontend JavaScript
    ├── style.css          # Styling
    └── backend/           # Node.js backend
        ├── server.js      # Express server
        ├── package.json   # Dependencies
        └── models/
            ├── Subject.js    # Subject model
            ├── Teacher.js    # Teacher model
            └── Timetable.js  # Timetable model
```

## Features
- Subject management
- Teacher management
- Timetable scheduling and assignment

## Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Models:** JavaScript data models for Subject, Teacher, and Timetable

## Installation & Setup
1. Navigate to the `backend` directory
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Open `index.html` in a web browser

## Usage
- Access the application through the web interface
- Manage subjects, teachers, and create timetables
- View generated schedules through the dashboard

## Output
Screenshots and visual demonstrations are available in the `Outputscreenshots` directory.
